# Diplom — программный комплекс для расчётов Паспорта безопасности объекта ТЭК

Проект: GUI (tkinter) + расчётные модули + генерация отчёта Word (docxtpl).

## Требования
- Python 3.9+ (Windows)
- Рекомендуется виртуальное окружение (venv)

## Установка
```powershell
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
